APS Scraper

A scraper that collects and displays the latest Australian Public Service job listings.

Interface or pipeline overview

Overview

The Australian Public Service advertises roles across many agencies. This project automates fetching those listings, normalising fields (title, agency, location, closing date), and presenting them in one place so you can browse or filter without visiting every site manually.

What I did

Implemented HTTP requests and HTML parsing with polite rate limits, handled pagination or search endpoints, and stored or rendered results in a simple web UI or export format.

Gallery

Listings view or CLI output

Tools & stack

Python with requests and Beautiful Soup; lightweight UI with Flask or Streamlit where used.
