Better Planner

A reminders and calendar application from my high school software project, built with Swift and SwiftUI.

Calendar or reminder list

Overview

Better Planner helps students keep assignments and events in one place: create reminders, see what’s due, and navigate a calendar tuned for school workflows. SwiftUI keeps the interface native and responsive on Apple platforms.

What I did

Modelled tasks and dates, implemented list and calendar views, and integrated local notifications with persistent storage for assignments and events.

Alternate tab, widget, or detail view

Tools & stack

Swift and SwiftUI in Xcode; persistence via UserDefaults or Core Data for local data.
